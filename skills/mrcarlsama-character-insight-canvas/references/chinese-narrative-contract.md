# 中文叙事契约

这份契约适用于简体中文产出。目标不是把英文分析术语换成中文词，而是让读者沿着中国当代非虚构写作熟悉的顺序认识角色：先看这个人身处什么局面、做了什么，再理解这些行动说明了什么。

自然化只能改写节奏和措辞，不能改动论断边界。归因、条件、冲突、版本范围和不确定性必须原样保留。

## 1. 先人后论

人物介绍按这个顺序组织：

1. 她是谁，眼下站在什么位置；
2. 她被什么事情牵住；
3. 她通常怎么做；
4. 这些做法带来什么后果；
5. 最后才给出解释。

不要一上来就写“她的核心弧线是……”“她象征着……”。抽象判断必须落在已经讲清楚的处境、动作或关系上。

生硬：

> 她的核心弧线是夺回主体性，并完成从历史容器到自我行动者的转化。

自然：

> 很长一段时间，她都在替别人留下的使命奔走。真正的变化，不是她突然变得善良，而是她开始自己决定要追什么，又愿意为这个决定付什么代价。

后一种写法仍然需要事实和解释分别挂到对应 claim，不能因为读起来顺就混成一个“文本确认”。

## 2. 写行动，不给性格贴空标签

“坚强、复杂、温柔、疯狂、有魅力”都太空。先写她在什么情况下做了什么，再用一句判断收束。

生硬：

> 她是一个兼具脆弱性与能动性的复杂角色。

自然：

> 她明知道这一步会把关系推向破裂，还是先做了决定，事后才解释。她不是不害怕，只是不肯把最后的选择交给别人。

同一性格至少要有行动、选择、反复行为或创作者说明作支撑。一个漂亮姿势不能证明一种人格。

## 3. 关系写“彼此怎么改变对方”

不要停在父女、宿敌、同伴、镜像、救赎者等标签。要说清：

- 对方让她看见了什么；
- 她在这段关系里想拿到什么；
- 两个人争的到底是什么；
- 相处之后，谁的选择发生了变化；
- 这段关系付出了什么代价。

如果两个人只是处境一正一反，就把正反关系讲出来，不必机械地称为“镜像”。如果作品没有明确写“救赎”，不要替角色安排救赎。

## 4. 核心路径写成“事情怎么一步步变了”

每一步优先回答四件事：

- 当时发生了什么；
- 哪个压力或关系进入场面；
- 角色自己做了什么选择；
- 此后什么已经不同。

只有来源明确建立因果，才可以写“导致、迫使、因此”。如果只是先后相接，写“随后、从这里开始、到这一步时”。如果是本报告的理解，写“可以看作、本报告据此理解为”，并保留 interpretation 标记。

## 5. 把限定词写进正文

条件和归因不能只躲在来源抽屉里。

- 角色自述：按她自己的说法……
- 他人判断：医疗人员据此判断……
- 条件事实：如果这段自述属实……
- 二手重建：现有场景记录显示……
- 解释：本报告更倾向把它理解为……
- 版本冲突：动画在这里改成了……，漫画则……

来源说“可能”，正文就不能写“必定”；来源只证明“她说过”，正文就不能替她盖章为客观事实。

## 6. 少用翻译腔和概念套话

下列写法不是绝对禁词，但若前文没有具体内容，通常应当重写：

- “这不是 A，而是 B”反复出现；
- “这意味着”“这表明”“换言之”连续起句；
- “作为……扮演着……角色”；
- “不仅仅是……更是……”；
- “核心弧线”“主体性诞生”“方向绑定”“问题前提失效”；
- “权力动态”“情感 stakes”“角色功能”“完成了……转化”；
- 三个结构完全相同的分句机械排比；
- 每句话都以“她”开头。

术语可以保留，但第一次出现时必须用具体中文解释：

| 分析术语 | 优先写法 |
| --- | --- |
| character arc | 这个人一路怎么变、变化脉络 |
| agency | 自己作主、选择权、由谁做决定 |
| stakes | 代价、利害、失去什么 |
| mirror character | 两个人处境哪里相反、彼此照见什么 |
| redemption | 只有证据成立时写救赎，否则写关系带来的具体变化 |
| motif | 反复出现的动作、物件、话语或画面 |

不要为了躲术语把文章切成电报句。短句和长句要有呼吸，段落围绕一个推进动作展开。

## 7. 首屏的中文节奏

首屏不是论文摘要。推荐次序：

1. 一句辨识度高、但不过度拔高的判断；
2. 两到三句交代身份、困境和最鲜明的做法；
3. “第一眼”写观众容易形成的印象；
4. “深入后”指出这个印象哪里不够；
5. 用一个清楚的动作进入核心路径。

一句话论点宜能被普通读者复述。介绍宜在手机首屏正常字号下读完，不要为了塞进抽屉而缩到小字。

## 8. 完稿自检

逐段检查：

- 删掉抽象名词后，读者还知道发生了什么吗？
- 判断前面是否已有动作或场景？
- 关系是否写出了双方的作用，而不只是标签？
- 句子是否像中文原生写作，而不是英文结构逐词替换？
- 是否为了“有力量”擅自加入必然、唯一、彻底、瓦解、救赎等强词？
- 条件、说话者和版本范围是否仍在正文里？
- 朗读时是否顺畅，还是一串名词在互相修饰？

自然不是口语化，更不是随意。好中文首先把人和事情说清楚，然后才谈概念。
